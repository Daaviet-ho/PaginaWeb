Política de Privacidad

Cómo tratamos tus datos personales

Responsable del tratamiento

Proyecto: DreamsDeveloper

Ubicación: Ibiza, Illes Balears, España

Contacto: A través del formulario de contacto del sitio web.

Datos que recogemos

A través de este sitio web podemos recoger los siguientes datos personales:

  • Formulario de contacto: nombre, dirección de correo electrónico, tipo de proyecto y mensaje.
  • Registro de usuario: nombre, correo electrónico y contraseña (almacenada de forma cifrada).
  • Navegación: datos técnicos como dirección IP, tipo de navegador y páginas visitadas, únicamente mediante cookies técnicas o de análisis anonimizado, cuando aplique.

Finalidad del tratamiento

Los datos personales recogidos se utilizan exclusivamente para:

  • Responder consultas y solicitudes realizadas a través del formulario de contacto.
  • Gestionar el acceso a áreas restringidas del sitio, cuando corresponda.
  • Comunicar información relacionada con presupuestos, proyectos o servicios solicitados.
  • Mejorar el funcionamiento y la experiencia de uso del sitio web.

Base jurídica

El tratamiento de datos se basa en:

  • Consentimiento del usuario al enviar el formulario de contacto o registrarse.
  • Ejecución de medidas precontractuales cuando el usuario solicita un presupuesto o información sobre un servicio.
  • Interés legítimo del responsable para el correcto funcionamiento técnico del sitio.

Conservación de los datos

Los datos se conservarán mientras sean necesarios para la finalidad para la que fueron recogidos, mientras el usuario mantenga su cuenta activa o mientras no se solicite su supresión. Una vez cumplida la finalidad, los datos se eliminarán o anonimizarán.

Cesión de datos a terceros

Los datos personales no se cederán a terceros, salvo obligación legal. Los datos del formulario de contacto pueden ser procesados por Netlify (proveedor de alojamiento) conforme a sus propias políticas de privacidad, como encargado del tratamiento.

Derechos del usuario

De acuerdo con la Ley Orgánica 3/2018 y el Reglamento (UE) 2016/679, el usuario puede ejercer en cualquier momento los siguientes derechos:

  • Acceso: conocer qué datos personales se están tratando.
  • Rectificación: corregir datos inexactos o incompletos.
  • Supresión: solicitar la eliminación de los datos cuando ya no sean necesarios.
  • Oposición: oponerse al tratamiento en determinadas circunstancias.
  • Limitación: solicitar la restricción del tratamiento cuando proceda.
  • Portabilidad: recibir los datos en un formato estructurado y de uso común.

Para ejercer cualquiera de estos derechos, el usuario puede enviar su solicitud a través del formulario de contacto, indicando claramente el derecho que desea ejercer y acompañando copia de un documento identificativo.

Asimismo, el usuario tiene derecho a presentar una reclamación ante la Agencia Española de Protección de Datos (www.aepd.es) si considera que sus datos no se están tratando correctamente.

Medidas de seguridad

DreamsDeveloper aplica medidas técnicas y organizativas adecuadas para proteger los datos personales contra accesos no autorizados, pérdida, alteración o destrucción. Las contraseñas se almacenan cifradas y las comunicaciones del sitio se realizan mediante protocolo HTTPS.
